获取方式

  1. 从 GitHub 仓库克隆本项目源码
  2. 下载发布的压缩包并解压
  3. 直接下载打包好的 exe 版本
前往 GitHub 下载

运行要求

  • 操作系统：Windows 8 / Server 2012 及以上
  • 联网环境（下载组件）
  • 未封禁的 CPUID / MAC 地址

推荐配置

  • JDK 8 或 OpenJDK 8
  • ≥ 2 核心 CPU（8 核以上最佳）
  • ≥ 2GB 内存
  • Python 3.6+
  • 有线网络连接
