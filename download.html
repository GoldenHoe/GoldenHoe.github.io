获取方式

  1. 从 GitHub 仓库克隆本项目源码
  2. 直接下载打包好的 exe 版本
  3. 点击下面的按钮直接下载或查看本项目的GitHub Releases
下载稳定版(v1.1.0.3) 下载Beta版(v1.2.0.0-Beta1) GitHub Releases

运行要求

  • 操作系统：Windows 8 / Server 2012 及以上
  • 联网环境

推荐配置

  • JDK 8 或 OpenJDK 8
  • ≥ 2 核心 CPU（8 核以上最佳）
  • ≥ 2GB 内存
  • Python 3.6+
  • 有线网络连接

如何使用？

  • 下载后直接打开exe可执行文件即可

Beta版注意

  • 现阶段Beta版(v1.2.0.0-Beta1)需要配合版本和URL文件使用，否则会报错
